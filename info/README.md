
## Features
Fitur Kantin Kejujuran:
- SigIn dan SignUp Siswa menggunakan ID dan Password
- Katalog Produk
- Detail Produk
- Akun Siswa 
    - Data Diri
    - Dompet (Informasi, Topup dan Withdraw Saldo)
    - Tambah Produk
    - List Produk
    - Riwayat Pembelian
    - Riwayat Penjualan
 - Pengurutan/Sorting Produk berdasarkan nama dan timestamp ditambahkan 
 - Flash Message
